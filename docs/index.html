About this map

ED-level Mayor primary results don't exist in any official BOE publication — BOE only releases citywide round-by-round PDFs and the raw cast vote records. This site computes per-ED first-choice and ranked-choice final-round tallies from the CVR ZIP files for each election. See methodology for sources, definitions, and limits.

Source: NYC Board of Elections CVR + NYC Open Data ED shapefile.
